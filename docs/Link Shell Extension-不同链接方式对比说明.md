# 智能复制和镜像以及Junction
| 方法           | 详细解释                                                                                                                                                                                                                                                               | 跨卷支持 | 文件/目录支持 | 链接类型 | 复制行为           |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---- | ------- | ---- | -------------- |
| Smart Copy   | 想象你有一个非常聪明的搬家公司。他们不仅会搬运你的所有物品,还会记住每件物品的原始摆放位置、它们之间的关系,甚至保留你那些特殊的收纳方式(比如用Hardlink存储的文件)。到了新家,他们会尽可能还原原来的摆设,只是地址变了。关于空间占用：Smart Copy 确实会占用额外空间，因为它创建了文件的实际副本。但它很"聪明"，会尽量保留原有的 Hardlink 结构，所以可能比普通复制占用更少的空间。就像聪明的搬家公司，会把原本用 Hardlink 存储的相同物品识别出来，在新家也用同样的方式存放，从而节省空间。 | 支持   | 文件和目录   | N/A  | 智能复制,保留链接结构    |
| Smart Mirror | 设想你有一面魔法镜子,它不仅能反映你房间的样子,还能实时更新。如果你在原房间里移动了家具,镜子里的影像也会跟着变。Smart Mirror就是这样,它创建了一个目录的实时副本,始终保持与源目录同步。关于空间占用：Smart Mirror 通常会占用与源目录相当的额外空间，因为它创建了一个完整的副本。但是，它也很"聪明"。如果源目录中有 Hardlink，Smart Mirror 会在镜像中保留这种结构，可能会比简单复制占用更少的空间。想象魔法镜子不仅复制了房间的样子，还复制了房间里所有的节省空间的小技巧。    | 支持   | 文件和目录   | N/A  | 智能复制并同步,保留链接结构 |
| Junction     | 把Junction想象成一个魔法门。你在房子(目录)的一个房间里安装了这个门,打开它就可以直接进入可能位于完全不同位置的另一个房间。这个魔法门占用的空间很小,但能让你瞬间到达目标位置。然而,如果目标房间被拆除了,这个魔法门就会失效。                                                                                                                                               | 支持   | 仅目录     | 软链接  | 不复制数据          |

注释说明：

1. 文件系统要求：以上所有方法都需要NTFS文件系统的支持。就像高级魔法需要特殊的魔法场地一样,这些高级文件操作需要NTFS这个"高级场地"来施展。

2. 链接类型：
   - 软链接（如Symbolic Link）：就像用细线连接两个物体。线很容易连接和断开,可以跨越远距离,但如果一端的物体移动了,线可能会断。
   - 硬链接（如Hardlink）：想象成用坚固的金属棒连接物体。它们牢固地绑在一起,移动一个会影响所有,但不能延伸太远（不能跨越磁盘分区）。

3. 跨卷支持：指的是这个"魔法"是否能跨越不同的"魔法领域"（磁盘分区或网络位置）使用。

4. 复制行为：
   - "不复制数据"就像用魔法创造分身,看起来有多个,但本质上是同一个。
   - "智能复制"就像有一个聪明的助手,它知道哪些需要真的搬运,哪些只需要留个记号。
   - "增量备份"就像一个细心的历史学家,只记录发生变化的部分,而不是每次都抄写整本书。

5. 使用这些方法时,要记住你是在操作强大的"文件魔法"。像所有强大的工具一样,要小心使用,特别是在处理重要文件时。在进行重大操作前,最好先进行备份,就像冒险前准备一个救生包一样。

6. 这些功能都是由Link Shell Extension这个"魔法工具箱"提供的。它就像是为Windows资源管理器添加了一套高级魔法装备,让你能够轻松地施展这些复杂的文件操作魔法。
